The Output Feedback (O\+FB) mode is a confidentiality mode that features the iteration of the forward cipher on an IV to generate a sequence of output blocks that are exclusive-\/\+O\+Red with the plaintext to produce the ciphertext, and vice versa. The O\+FB mode requires that the IV is a nonce, i.\+e., the IV must be unique for each execution of the mode under the given key. Refer to SP 800-\/38A for more details
