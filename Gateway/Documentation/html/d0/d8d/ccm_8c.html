Description

CCM mode (Cipher Block Chaining-Message Authentication Code) is a mode of operation for cryptographic block ciphers. It is an authenticated encryption algorithm designed to provide both authentication and confidentiality. CCM mode is only defined for block ciphers with a block length of 128 bits. Refer to SP 800-38D for more details
