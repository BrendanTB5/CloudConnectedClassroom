Description

The TCP-MIB describes managed objects used for implementations of the Transmission Control Protocol (TCP) in an IP version independent manner. Refer to the following RFCs for complete details:

  • RFC 4022: MIB for the Transmission Control Protocol (TCP)
  • RFC 4001: Textual Conventions for Internet Network Addresses
